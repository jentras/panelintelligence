How this was made

This project is an open-source knowledge garden for event organizers and nonprofits. It supports speaker uploads, bidirectional links, graph views, and share-ready quote cards.

Architecture

  • Static front-end in index.html
  • Node HTTP backend in server.js
  • JSON data store in data/*.json
  • Link reindex job at POST /api/links/reindex
  • Obsidian export at GET /api/export/obsidian

Open-source replication guide

  1. npm install
  2. npm start
  3. Upload speakers via POST /api/speakers or /api/speakers/import
  4. Run POST /api/links/reindex to build backlinks
  5. Generate quote cards with POST /api/quotes/:id/generate-card

Data ethics and consent

Only publish approved quotes and media. Keep consent flags for speaker promotional assets and include removal workflows for nonprofit compliance.
